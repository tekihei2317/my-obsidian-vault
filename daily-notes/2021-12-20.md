こういうクエリで内側の`id`はどっちのテーブルの`id`を指すかが分からなかった。内側が優先になるっぽい（下の例でいうと`roles.id`を指している）。

```sql
select *
from `users`
where
  exists (
    select *
    from `roles`
    where
      `users`.`role_id` = `roles`.`id` and `id` = 5
  )
;
```
